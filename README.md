## 简述
弹幕文本分析以及语言模型项目

## 命名规则
X/Y/train/test_Class_Description 或 Class_Description   
含义：     
- X：特征数据集，Y：标签，train：训练集，test：测试集
- Class： 变量名称， Description：变量作用或来源简介
- 逐步改变的变量，则后加 _Feature， 表示当前改变的来源特征
- 以 _ 开头的变量表示不可变
